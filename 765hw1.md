1. Two Riemannian metrics $g$ and $h$ on the same manifold $M$ are said to be conformal if there exists a positive smooth function $f(x)$ such that
  $g=f \cdot h .$
  Show that angles between tangent vectors are the same with respect to $g$ and $h$.
2. Show that the round metric on $S^n$ is given in stereographic coordinates by
    $$g_{i j}=\frac{4 \delta_{i j}}{\left(1+|x|^2\right)^2} .$$
    For the unit n-sphere $S^n$ embedded in $\mathbb{R}^{n+1}$ with the standard Euclidean metric, the inverse stereographic projection $\Phi: \mathbb{R}^n \to S^n \subset \mathbb{R}^{n+1}$ from the North Pole $(0,\dots,0,1)$ is given by
    $u^i = \frac{2x^i}{1+|x|^2} \quad \text{for } i=1,\dots,n,$
    $u^{n+1} = \frac{|x|^2 - 1}{|x|^2 + 1} = 1 - \frac{2}{1+|x|^2},$
    where $|x|^2 = \sum_{k=1}^n (x^k)^2$.
    When we use stereographic coordinates $x^{i}$, we are defining an embedding map $\Phi :\mathbb{R}^{n}\rightarrow \mathbb{R}^{n+1}$ such that each ambient coordinate $u^{k}$ is a function of the local coordinates: $u^{k}=u^{k}(x^{1},\dots ,x^{n})$. The differential of each ambient coordinate is: $`du^{k}=\sum _{i=1}^{n}\frac{\partial u^{k}}{\partial x^{i}}dx^{i}`$. Substitute these differentials back into the Euclidean line element $ds^{2}$: $`ds^{2}=\sum _{k=1}^{n+1}\left(\sum _{i=1}^{n}\frac{\partial u^{k}}{\partial x^{i}}dx^{i}\right)\left(\sum _{j=1}^{n}\frac{\partial u^{k}}{\partial x^{j}}dx^{j}\right)`$. Rearranging the sums to group the $dx^{i}dx^{j}$ terms: $`ds^{2}=\sum _{i,j=1}^{n}\left(\sum _{k=1}^{n+1}\frac{\partial u^{k}}{\partial x^{i}}\frac{\partial u^{k}}{\partial x^{j}}\right)dx^{i}dx^{j}`$. By definition, the components of the metric tensor $g_{ij}$ are the coefficients of $dx^{i}dx^{j}$ in the line element $ds^{2}=\sum g_{ij}dx^{i}dx^{j}$. Thus: $`g_{ij}=\sum _{k=1}^{n+1}\frac{\partial u^{k}}{\partial x^{i}}\frac{\partial u^{k}}{\partial x^{j}}`$.
    The pullback of the round metric on $S^n$ to stereographic coordinates on $\mathbb{R}^n$ yields the induced metric tensor
    $g_{ij} = \frac{4 \delta_{ij}}{(1 + |x|^2)^2}.$
    This metric is conformal to the flat Euclidean metric on $\mathbb{R}^n$ with conformal factor $2 / (1 + |x|^2)$, because the stereographic projection is a conformal map, so the line element is
    $ds^2 = \frac{4 \sum_{i=1}^n (dx^i)^2}{(1 + |x|^2)^2}.$
    It is derived by computing the partial derivatives of the embedding $\Phi$: Let $D = 1 + |x|^2$. Then
    $\frac{\partial u^k}{\partial x^i} = \frac{2 \delta_{ki} D - 4 x^k x^i}{D^2} \quad \text{for } k \leq n,$
    $\frac{\partial u^{n+1}}{\partial x^i} = \frac{4 x^i}{D^2}.$
    The metric components are $g_{ij} = \sum_{k=1}^{n+1} \frac{\partial u^k}{\partial x^i} \frac{\partial u^k}{\partial x^j}$, which expands to
    $g_{ij} = \sum_{k=1}^{n} \left( \frac{2 \delta_{ki} D - 4 x^k x^i}{D^2} \right) \left( \frac{2 \delta_{kj} D - 4 x^k x^j}{D^2} \right) + \frac{16 x^i x^j}{D^4}.$
    Expanding the summation yields
    $g_{ij} = \frac{1}{D^4} \left( 4 \delta_{ij} D^2 - 16 x^i x^j D + 16 x^i x^j (|x|^2 + 1) \right).$
    Since $D = 1 + |x|^2$, the cross terms cancel: $-16 x^i x^j D + 16 x^i x^j (|x|^2 + 1) = -16 x^i x^j (1 + |x|^2) + 16 x^i x^j (1 + |x|^2) = 0$, leaving
    $g_{ij} = \frac{4 \delta_{ij} D^2}{D^4} = \frac{4 \delta_{ij}}{(1 + |x|^2)^2}.$
